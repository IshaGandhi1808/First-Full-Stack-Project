# 🌍 Wanderlust

Wanderlust is a full-stack Airbnb-inspired web application that allows users to explore, create, and review travel destinations with an interactive and user-friendly interface.

---

## 🚀 Live Demo

👉 https://wanderlust-m4u6.onrender.com

---

## ✨ Key Features

### 🔐 Authentication & Authorization

- User signup and login
- Secure session-based authentication
- Only listing owners can edit or delete their listings

### 🏠 Listings Management

- Create new travel listings
- View all listings
- Update existing listings
- Delete listings

### ⭐ Reviews System

- Add reviews to listings
- Delete reviews
- User-specific review control

### 🔍 Search & Filters

- Search destinations by name
- Filter listings by categories

### 🗺️ Interactive Maps

- Integrated maps using MapLibre
- Location visualization with OpenStreetMap

### ☁️ Image Upload

- Upload and manage images using Cloudinary

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript, EJS
- Backend: Node.js, Express.js
- Database: MongoDB
- Services: Cloudinary, MapLibre, OpenStreetMap
- Deployment: Render

---

## 📌 Project Overview

This project demonstrates a complete full-stack application with CRUD functionality, user authentication, third-party integrations, and cloud deployment. It reflects real-world application architecture similar to platforms like Airbnb.
